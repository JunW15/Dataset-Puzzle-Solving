1. Description
In a possible crossword for Robots, letters are substituted with digits.
Each ‘word’ is formed by an uninterrupted sequence of numbers (i.e. 2-3-4-5), but in any order (i.e. 3-4-2-5)
1. Tips
1. Crossings will tell you sometimes which number can’t go in a ‘word’
2. Small words aren’t always easier.
