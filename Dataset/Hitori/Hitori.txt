1.@Summary
Darken some tiles so no number appears in a row or column more than once.
2.Description
The goal is to shade squares so that a number appears only once in a row or column.
While doing that, you must take care that shaded squares don't touch horizontally or vertically between them.
3.Tips
a.Hitori can be solved in different ways, usually start where a number appears clustered.
b.'L' shaped areas with the same number are sometimes a good starting point.
